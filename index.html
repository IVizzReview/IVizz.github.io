บทคัดย่อ (Abstract)

เนื่องจากในปัจจุบันปัญหาเด็กจมน้ำนั้นมีอัตราที่เพิ่มขึ้นทุกปี และเด็กไทยเสียชีวิตจากการจมน้ำสูงเป็นอันดับหนึ่งเมื่อเทียบกับอุบัติเหตุอื่น ๆ ซึ่งสาเหตุหลักของอุบัติเหตุนี้คือ ผู้ปกครองของเด็กไม่ได้ดูแลเด็กอย่างใกล้ชิด ทั้งอุปกรณ์ช่วยป้องกันการจมน้ำในปัจจุบันมีขนาดใหญ่และไม่เหมาะสำหรับใส่พกพาไปยังสถานที่อื่น ทำให้เมื่อเกิดอุบัติเหตุจมน้ำ ผู้ปกครองไม่สามารถเข้าช่วยเหลือเด็กได้ทันท่วงที ส่งผลให้เด็กมีโอกาสรอดน้อยลง และเสียชีวิตในที่สุด ผู้พัฒนาจึงได้คิดค้นอุปกรณ์ช่วยป้องกันและเพื่อเพิ่มโอกาสรอดของเด็กจากการจมน้ำ โดยอุปกรณ์นี้จะช่วยแจ้งเตือนผู้ปกครองของเด็ก ทันทีที่เมื่ออุปกรณ์ตรวจวัดแรงดันน้ำรอบตัวของเด็กอยู่ในระดับที่สูงผิดปกติ อุปกรณ์จะทำการส่งข้อความแจ้งเตือนไปยังโทรศัพท์ของผู้ปกครอง ทำให้ผู้ปกครองรู้ตัวและสามารถเข้าช่วยเหลือเด็กได้ทันท่วงที

Arduino Nano 3.0 Mini USB

Arduino Nano 3.0 เหมือน Arduino Nano 3.0 รุ่นเดิมทุกอย่าง แต่ราคาถูกกว่า ต่างกันที่ใช้ชิฟ CH340G เป็นชิฟ USB Driver แทนรุ่น Arduino Nano 3.0 เดิม สามารถใช้ได้กับ Windows XP, Windows 7, Windows 8, Windows 10 ทั้ง 32/64bit, Linux, Mac OS ได้ และมีราคาถูกลง

SIM900A

SIM900A GSM/GPRS Module โมดูล SIM900 เป็นโมดูลซิมการ์ดที่สามารถใส่ซิมการ์ด มือถือ ควบคุมการ โทรเข้า โทรออก รับสาย วางสาย ส่งsms ได้ ใช้ไฟเลี้ยง 5V 2A
ขนาด: 5×5cm
อุณหภูมิในการทำงาน: -40°C ~ 85°C
น้ำหนัก: 3.4g
ช่วงแรงดันไฟฟ้า: 5V
การใช้พลังงานต่ำ: 1.5mA (sleep mode)

Pressure Transducer Sensor

เซ็นเซอร์ใช้ไฟเลี้ยง 5V ให้สัญญาณออกมาเป็น analog ใช่้งานง่ายเชื่อมต่อ arduino ได้โดยตรง ทำจากเหล็กแข็งแรง ตัวเซ็นเซอร์กันน้ำ
สามารถวัดความดันได้: 0-1.6MPa
ผู้ผลิต: DFRobot
อุณหภูมิในการทำงาน: -20°C ~ 85°C
ความดันการดำเนินงาน: 232.06PSI (1600 kPa)
ความดันสูงสุด: 290.08PSI (2000 kPa)
